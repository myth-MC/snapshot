Snapshot is a tool designed to quickly collect key server data for troubleshooting. By generating a unique debug code, it simplifies the process of identifying and resolving issues with your server's configuration, plugins, and settings.

üîê Privacy Assurance

Data is only kept in our servers for 24 hours to ensure your information is handled responsibly and securely. After this period, all logs and related data are deleted.

During these 24 hours, we store the following data:

  • Server IP and port (hidden, temporarily stored for rate limiting purposes)
  • Plugin version and server version
  • List of active plugins (you can opt out)
  • Relevant plugin settings (you can opt out)
  • Timestamp of when this data was collected
